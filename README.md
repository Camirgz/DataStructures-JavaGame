#Proyecto Programado #03 - CI0112

##Descripción

Este proyecto implementa una serie de estructuras de datos y algoritmos en Java. Incluye una matriz de elementos, una lista doblemente enlazada y un árbol binario. El programa permite al usuario realizar varias operaciones con estas estructuras de datos a través de un menú interactivo.

##Clases y Paquetes

- `main.Main`: Contiene el punto de entrada del programa y el menú interactivo.
- `juego.Elemento`: Define los elementos de la matriz.
- `juego.MatrizDeJuego`: Maneja la creación y operaciones básicas de la matriz.
- `juego.Nodo`: Define un nodo para la lista doblemente enlazada.
- `juego.ListaDobleEnlazada`: Implementa la lista doblemente enlazada y sus operaciones.
- `juego.NodoArbol`: Define un nodo para el árbol binario.
- `juego.ArbolBinario`: Implementa el árbol binario y sus operaciones.

## Requisitos

- Java Development Kit (JDK) 8 o superior.

## Uso del Programa

Al ejecutar el programa, se presentará un menú con las siguientes opciones:

1. Crear/cargar – Guardar matriz
2. Crear Lista Enlazada Doble
3. Crear Árbol Binario
4. Operaciones con la Lista Enlazada Doble
5. Operaciones con el Árbol Binario
6. Salir

### Opciones del Menú

#### 1. Crear/cargar – Guardar matriz

1a. Crear nueva matriz: Crea una nueva matriz de juego.  
1b. Guardar matriz en archivo CSV: Guarda la matriz actual en un archivo CSV.  
1c. Cargar matriz desde archivo CSV: Carga una matriz desde un archivo CSV (a implementar).

#### 2. Crear Lista Enlazada Doble

Crea una lista doblemente enlazada a partir de los elementos de la matriz que cumplen ciertas condiciones (por ejemplo, valores impares).

#### 3. Crear Árbol Binario

Crea un árbol binario a partir de los elementos de la matriz.

#### 4. Operaciones con la Lista Enlazada Doble

Permite realizar diversas operaciones con la lista doblemente enlazada (a implementar).

#### 5. Operaciones con el Árbol Binario

Permite realizar diversas operaciones con el árbol binario (a implementar).

#### 6. Salir

Termina la ejecución del programa.
